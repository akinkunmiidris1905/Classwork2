Introduction to Git

Git is the open source version control system that is used to track changes made in our code

Advantages of using git

  • Speed and Performance
  • Distributed version control
  • Colaboration
  • Conflict Resolution
  • History and Tracability
  • Branching and Merging

Configuring Git

git config --global user.name "Akin Idris"

git config --global user.email "oluwabreinard@gmail.com"

Basic Git Command

  1. git init: This is used to initialize a git repository
  2. git status: This is used to check the status of our file
  3. git add . : This is used to track the changes made to our file
  4. git commit --m "the actual commit message" : This is used to save a snapshotof our file using descriptive caption
  5. git remote add origin: This is used to connect a local repository to a remote repository on git hub
  6. git push origin main: This is used to push code from local to repository
